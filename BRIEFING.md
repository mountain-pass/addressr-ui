# Project Briefing

## What You Need to Know

- **React component library** for Australian address autocomplete
- **Two main exports**: `useAddressSearch` hook (headless) + `AddressAutocomplete` component (styled)
- **HATEOAS via @windyroad/fetch-link** — follows RFC 8288 Link headers, never hardcodes URL paths
- **downshift** for accessible combobox pattern
- **CSS Modules** for styling — consumers import `@mountainpass/addressr-react/style.css`
- **Default API**: RapidAPI (`addressr.p.rapidapi.com`) with `x-rapidapi-key` auth
- **Default branch: `main`**
- **Published to npm** as `@mountainpass/addressr-react`
